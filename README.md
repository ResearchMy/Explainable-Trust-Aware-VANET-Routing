# Intelligent Explainable Trust-Aware Secure Routing for VANETs Using XGBoost–ABC Optimization

## Overview
This repository contains the implementation of an Explainable Trust-Aware Secure Routing Framework for VANETs using XGBoost, SHAP, and Artificial Bee Colony (ABC) optimization.

## Dataset
Dataset: CICIoT2023

## Methodology
1. Data Preprocessing
2. XGBoost Intrusion Detection
3. SHAP Explainability
4. Trust Score Generation
5. VANET Graph Construction
6. ABC Route Optimization

## Results
- Accuracy: 99.70%
- Precision: 99.93%
- Recall: 99.48%
- F1-Score: 99.70%
- ROC-AUC: 99.97%

## Reproducibility
Run the notebook or Python scripts in sequence:
1. preprocessing
2. xgboost_ids
3. shap_analysis
4. trust_generation
5. routing_framework
6. abc_optimization
